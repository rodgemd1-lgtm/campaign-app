# AI Resume Scorer

Instant AI-powered resume analysis to help optimize your resume for ATS systems and increase interview chances.

## Features

- 📄 Upload PDF/DOCX resumes
- 🤖 AI-powered ATS scoring (0-100)
- 💡 Detailed feedback on strengths and weaknesses
- 🎯 Actionable recommendations for improvement
- 💳 Stripe payment integration
- 📱 Responsive design

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Supabase for backend
- OpenAI for AI analysis
- Stripe for payments
- Framer Motion for animations

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   ```bash
   cp .env.local.example .env.local
   # Fill in your API keys
   ```

3. Run development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)

## Usage

1. Upload your resume (PDF or DOCX)
2. Click "Analyze Resume"
3. Get instant feedback and score
4. Upgrade for unlimited analysis
